# Happy-life
Flexi time for Happy-life company

Install TypeSpript:
1) npm install typescript --save-dev


Install NodeJS:
1) npm install
2) bower install

Running project:
1) ng build
2) ng serve
3) open http://localhost:4200/

Running e2e tests locally:
1) Start the server
2) Start the selenium server: webdriver-manager start
3) protractor protractor.conf.js
